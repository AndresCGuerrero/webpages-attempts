INTRODUCCIÓN

El hardware es la parte física de un sistema informático. Se compone de todos los componentes eléctricos, electrónicos, electromecánicos y mecánicos que permiten que el sistema funcione. El hardware se divide en dos categorías principales:

  • Periféricos: Son los dispositivos que se conectan al sistema informático, como el monitor, el teclado, el ratón, la impresora, etc.
  • Componentes internos: Son los dispositivos que se encuentran dentro del sistema informático, como la placa base, la memoria RAM, el disco duro, etc.

Periféricos

Los periféricos son los dispositivos que se conectan al sistema informático para permitir la entrada y salida de datos. Los principales tipos de periféricos son los siguientes:

  • Entrada: Son los dispositivos que permiten introducir datos al sistema informático, como el teclado, el ratón, el escáner, etc.
  • Salida: Son los dispositivos que permiten mostrar los datos del sistema informático, como el monitor, la impresora, los altavoces, etc.

Componentes internos

Los componentes internos son los dispositivos que se encuentran dentro del sistema informático y permiten que el sistema funcione. Los principales tipos de componentes internos son los siguientes:

  • Placa base: Es la base del sistema informático y conecta todos los demás componentes.
  • Microprocesador: Es el cerebro del sistema informático y se encarga de procesar los datos.
  • Memoria RAM: Es la memoria de trabajo del sistema informático y se utiliza para almacenar datos temporalmente.
  • Disco duro: Es la memoria de almacenamiento del sistema informático y se utiliza para almacenar datos de forma permanente.
  • Tarjeta gráfica: Es la tarjeta que se encarga de generar la imagen que se muestra en el monitor.
  • Tarjeta de sonido: Es la tarjeta que se encarga de generar el sonido que se escucha a través de los altavoces.
  • Tarjeta de red: Es la tarjeta que se encarga de conectar el sistema informático a una red.
